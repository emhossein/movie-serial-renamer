# Movie and Serial Renamer
The Movie and Serial Renamer is a simple desktop application built using Python and Tkinter. It is designed to help users quickly and easily rename their movie and serial files in a standardized format.

## Features
The application offers the following features:

- Choose the directory containing the files to be renamed
- Specify the file types (e.g. mp4, avi, etc.) to be renamed
- Add a prefix to the file name (optional)
- Rename the files in a standardized format: SXXEXX for serials, and just the file name for movies

## Getting Started
To use the Movie and Serial Renamer application, follow these steps:

1. Clone the repository to your local machine
2. Ensure that you have Python 3.x and Tkinter installed on your machine
3. Run the `python renamer.py` script to launch the application
4. Click the "Choose Folder" button to select the directory containing the files to be renamed
5. Enter the file types you wish to rename (e.g. mp4, avi) in the File Types entry field
6. Optionally, enter a prefix to be added to the file names in the Prefix entry field
7. Click the "Rename Files" button to rename the files in the specified format

## Limitations
The application has the following limitations:

- The renaming format is fixed and cannot be customized
- Only files with specified file types will be renamed
- The application does not have an undo function
